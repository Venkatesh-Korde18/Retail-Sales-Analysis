# Dataset Description

This dataset contains retail sales transaction data used for sales analysis and business intelligence.

It includes more than 10,000 transaction records with details related to orders, customers, products, sales, profit, discounts, shipping, and regional performance across different cities and states in the United States.

## Dataset Includes
- Order and shipping details
- Customer information
- Product categories and sub-categories
- Sales and profit data
- Regional and city-wise sales records

## Project Purpose
This dataset is used for:
- SQL analysis
- Sales and profit analysis
- Dashboard creation using Power BI and Tableau
- Excel-based data cleaning and reporting

## Tools Used
- MySQL
- Microsoft Excel
- Power BI
- Tableau
