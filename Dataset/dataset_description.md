# Dataset Description

This dataset contains retail sales transaction data used for business intelligence and sales analytics.

## Dataset Overview

The dataset includes over 10,000 retail transaction records containing customer, product, sales, shipping, and regional information across multiple locations in the United States.

## Key Columns Included

* Order ID
* Order Date
* Ship Date
* Customer Information
* Segment
* Country/Region
* City and State
* Product Category and Sub-Category
* Product Name
* Sales
* Quantity
* Discount
* Profit

## Business Analysis Scope

This dataset is used to perform:

* Sales KPI analysis
* Regional sales performance analysis
* Customer segmentation analysis
* Product category performance evaluation
* Profit and discount analysis
* SQL-based business querying
* Dashboard development using Power BI and Tableau
* Excel-based data cleaning and reporting

## Tools Used

* MySQL
* Microsoft Excel
* Power BI
* Tableau

## Project Objective

The objective of this project is to analyze retail business performance and generate actionable insights using data analytics and business intelligence tools.
